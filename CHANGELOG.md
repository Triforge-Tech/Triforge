# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-04

### Added
- Initial release
- Homepage with hero, features, recent posts, and CTA sections
- About page with company story, values, and team
- Services page with detailed service offerings
- Blog listing with search, filters, and pagination
- Blog post pages with rich text, code highlighting, TOC, and social sharing
- Contact page with validated form
- MDX + Contentlayer for content management
- Dark mode support
- SEO optimization with meta tags and Open Graph
- Sitemap and robots.txt generation
- Mobile-responsive design
- Accessible (WCAG 2.1 AA)

- MDX + Contentlayer for content management
- Framer Motion animations
- Code syntax highlighting
- Reading time estimation
- Category filtering
- Blog search
- Newsletter signup form
- Contact form with validation
- Social sharing buttons
- Table of contents for blog posts
- Related posts
- Author profiles
- Image optimization

### Performance

- Fast loading times
